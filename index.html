Solving a System of Differential Equations

2x2 System — Substitution with Parameters Variation Method

The Problem

Part A - Find a solution of the homogeneous system with a tα component:
dx/dt = (3/t)x − 4ty
dy/dt = (1/t3)x + (5/t)y
Part B - Using the solution you found, find all solutions of:
dx/dt = (3/t)x − 4ty + 5t3
dy/dt = (1/t3)x + (5/t)y − 4t

Solution Method

A system with variable coefficients. We find a homogeneous solution using a power-of-t ansatz, then use reduction of order (change of variables) to find the general solution of the non-homogeneous system.

Step 1 Solution - Homogeneous Solution via Power Ansatz

Step a: Guessing the solution form
x = atα,   y = btβ
Step b: Substitution into the first equation
αatα−1 = (3/t)(atα) − 4t(btβ)
αatα−1 = 3atα−1 − 4btβ+1
Need: α − 1 = β + 1, so α = β + 2
Step c: Substitution into the second equation (verification)
βbtβ−1 = (1/t3)(atα) + (5/t)(btβ)
Need: α − 3 = β − 1, so α = β + 2 ✓
Step d: Choosing β = 3, α = 5
From eq 1: 5a = 3a − 4b ⇒ 2a = −4b ⇒ a = −2b
From eq 2: 3b = a + 5b ⇒ a = −2b ✓
Choose b = 1: a = −2

The Homogeneous Solution

φ(t) = [−2t5, t3]T

Step 2 Solution - Change of Variables (Reduction of Order)

Step a: Defining the substitution
x = ξ − 2t5η,   y = t3η
Step b: Differentiation using the product rule
dx/dt = dξ/dt − 10t4η − 2t5(dη/dt)
dy/dt = 3t2η + t3(dη/dt)
Step c: Substitution into the system and simplification
After substitution and cancellation of the homogeneous terms (which cancel thanks to the homogeneous solution),
two separate scalar equations are obtained

The Scalar Equations

dξ/dt = (5/t)ξ − 3t3
dη/dt = t−6ξ − 4t−2
Key insight:
The system "decouples" - first solve the ξ equation (alone), then substitute ξ into the η equation.

Step 3 Solution - The ξ Equation

The equation:
dξ/dt − (5/t)ξ = −3t3
Integrating factor:
μ = e−∫(5/t)dt = e−5 ln t = t−5
Multiply by integrating factor:
d/dt(t−5ξ) = −3t−2
Integration:
t−5ξ = −3 · ∫t−2dt = 3t−1 + A
ξ = 3t4 + At5

The Solution

ξ = 3t4 + At5

Step 4 Solution - The η Equation

Substituting ξ into the equation:
dη/dt = t−6(3t4 + At5) − 4t−2
Simplification:
dη/dt = 3t−2 + At−1 − 4t−2
dη/dt = −t−2 + At−1
Term-by-term integration:
η = ∫(−t−2)dt + A·∫(t−1)dt + B
η = t−1 + A·ln(t) + B
Note:
A is the same constant from the ξ solution, and B is a new integration constant.

The Solution

η = t−1 + A·ln(t) + B

Step 5 Solution - The General Solution

Back-substitution for x:
x = ξ − 2t5η
= (3t4 + At5) − 2t5(t−1 + A·ln t + B)
= 3t4 + At5 − 2t4 − 2At5·ln t − 2Bt5
= t4 + A(t5 − 2t5·ln t) − 2Bt5
Back-substitution for y:
y = t3η = t3(t−1 + A·ln t + B)
= t2 + At3·ln t + Bt3
Identifying the particular solution (A=B=0):
xp(t) = t4,   yp(t) = t2

The General Solution

x(t) = t4 + A(t5 − 2t5·ln t) − 2Bt5
y(t) = t2 + At3·ln t + Bt3
In vector form:
[x(t); y(t)] = [t4; t2] + A·[t5−2t5ln t; t3ln t] + B·[−2t5; t3]
